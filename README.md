
## Download project
        -git clone https://github.com/MediumBass/superHeroes
        -cd superHeroes
# Start Database:
        -docker-compose up
# Start Backend:
        -cd back
        -npm i
        -npm start
# Start tests: 
        -cd back
        -npm test
(Tests will also fill Database with starting data)
# Start Frontend
        -cd front
        -npm i
        -npm start
# The application will run on 
        http://localhost:3000/
